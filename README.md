# cangjie.vim

使 `vim` 支持 `cangjie` 语法

- [ ] 语法高亮
- [ ] LSP Server

## 使用方法
### 使用插件管理器
```vim
Plug 'https://gitcode.com/Neila/cangjie.vim.git'
or
Plugin 'https://gitcode.com/Neila/cangjie.vim.git'
```
